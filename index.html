🛠️ Crear un calendari compartit amb filtres per categoria a Airtable

1. Crear una base nova

  • Accedeix a airtable.com i inicia sessió o crea un compte gratuït.

  • Fes clic a "Add a base" i selecciona "Start from scratch".

  • Dona-li un nom, com ara "Calendari Compartit".

2. Configurar la taula

  • A la taula principal, modifica les columnes per incloure:

    • Títol (Single line text)

    • Data d'inici (Date)

    • Data de finalització (Date, opcional)

    • Categoria (Single select): afegeix opcions com "Reunió", "Taller", "Lliurament", etc.

    • Descripció (Long text)

    • Ubicació (Single line text, opcional)

3. Afegir la vista de calendari

  • Fes clic a "Grid view" a la part superior esquerra i selecciona "Calendar".

  • Assigna la Data d'inici com a camp de data principal.

  • Si vols mostrar intervals, pots configurar la Data de finalització com a camp de finalització.

4. Aplicar filtres per categoria

  • A la vista de calendari, fes clic a "Filter".

  • Afegeix una condició: "Categoria" és igual a la categoria desitjada (per exemple, "Reunió").

  • Pots crear múltiples vistes de calendari, cadascuna amb un filtre diferent per categoria.

5. Compartir el calendari

  • Fes clic a "Share view" a la part superior dreta.

  • Activa l'opció "Share publicly" i copia l'enllaç generat.

  • Comparteix aquest enllaç amb els agents que necessiten visualitzar el calendari. No necessiten un compte d'Airtable per accedir-hi.
